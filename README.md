A Symfony bundle for including the [Fitbit SDK for PHP](https://github.com/jbtcd/fitbit-sdk-php).

## Installation

To install, use composer:

```bash
$ composer require jbtcd/fitbit-bundle
```

## Testing

``` bash
$ composer run tests
```
